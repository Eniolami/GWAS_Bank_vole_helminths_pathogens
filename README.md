# GWAS_Bank_vole_helminths_pathogens
This repository contains all datasets and R codes used to generate the results in the article "Genome-wide associations of host susceptibility to helminth and blood pathogens in a spatially structured rodent populations" by Olarewaju et al.  

The Variant Call Format file contains Genotype information for 182 Bank vole individuals (32000 SNPs) used as an input file for the GWAS analysis after quality filtering
